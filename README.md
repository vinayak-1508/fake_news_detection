Fake News Detection Using Machine Learning

This project focuses on detecting fake news using machine learning algorithms. The model is trained on a dataset containing news articles labeled as real or fake. The main goal is to classify news based on the text content.

Features

Text preprocessing and cleaning

TF-IDF vectorization

Machine learning models for classification

Evaluation using accuracy and confusion matrix

Models Used

Logistic Regression

Passive Aggressive Classifier

Naive Bayes

How to Run

Download the project or clone the repository.

Open the Jupyter Notebook:
FAKE_NEWS_DETECTION_USING_ML.ipynb

Run all cells to train and test the models.

Requirements

Python

Jupyter Notebook

scikit-learn

pandas

numpy

Dataset

The project uses a fake news dataset containing text and labels (real/fake).

Output

The notebook shows model accuracy and predictions for fake or real news.
